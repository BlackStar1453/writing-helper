## 设计原则记录

### 字体系统一致性
- **中文**: Noto Serif SC，大字号加粗 (饱和感)
- **英文**: Poppins，相对较小精致 (点缀作用)
- **图标**: Font Awesome 线稿风格

### 四段式布局严格执行
1. **页头**: 专业刊头，主副标题 + 版本信息
2. **主体**: 4+8 非对称网格，视觉锚点 + 核心信息
3. **中段**: 全宽分隔区，不同背景色，展示对比数据
4. **页脚**: 深色背景，强烈对比，联系信息

### 视觉锚点设计
- **尺寸**: 12rem (192px)
- **样式**: 空心描边，无填充
- **位置**: 4列侧边栏居中
- **作用**: 创造压倒性视觉冲击力

### 响应式适配
- **主要适配**: 1200x1600 宽高比
- **网格系统**: Tailwind CSS 12列网格
- **断点**: md (768px) 以上保持设计完整性
